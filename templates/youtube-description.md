# YouTube Description Template

## Platform Constraints
- First 150 characters shown in search results — lead with value
- Max 5,000 characters total
- Timestamps: include even as placeholders for later editing
- Tags: 5-10 relevant terms at the end

## Format

```
[OPENING 150 CHARS — must describe value without needing context]

[Blank line]

In this video: [1-2 sentence expansion]

[Blank line]

KEY TAKEAWAYS:
• [Takeaway 1]
• [Takeaway 2]
• [Takeaway 3]
• [Takeaway 4]
• [Takeaway 5]

[Blank line]

TIMESTAMPS:
00:00 - Introduction
[XX:XX - Chapter title]
[XX:XX - Chapter title]
[XX:XX - Conclusion]

[Blank line]

LINKS & RESOURCES:
→ Full article: [URL]
→ [Related resource]: [URL]

[Blank line]

ABOUT [CHANNEL/AUTHOR]:
[1-2 sentence bio]

[Blank line]

TAGS: tag1, tag2, tag3, tag4, tag5
```

## Tips
- Search engines index YouTube descriptions — include natural keywords
- Timestamps improve watch time significantly
- Link to related content to reduce subscriber churn
