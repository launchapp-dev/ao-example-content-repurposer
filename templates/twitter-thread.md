# Twitter/X Thread Template

## Platform Constraints
- Each tweet: max 280 characters (including spaces)
- Thread length: 5-12 tweets
- Thread numbering: use "1/" format
- No bare links in mid-thread tweets (save for final CTA tweet)

## Format

```
1/
[HOOK TWEET — bold claim, surprising stat, or compelling question]
[Max 280 chars]

2/
[Context or problem statement]
[Max 280 chars]

3/
[Key insight #1]
[Max 280 chars]

[Continue for each major insight...]

[FINAL TWEET]
[CTA — "Full article:" + URL]
[Max 280 chars]
```

## Tips
- Put the most surprising stat in tweet 1 or 2
- Use line breaks within tweets for readability (not em-dashes)
- Don't end a tweet with a cliff-hanger that requires reading the next
- Each tweet should stand alone as a useful thought
