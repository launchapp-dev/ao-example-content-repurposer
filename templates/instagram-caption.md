# Instagram Caption Template

## Platform Constraints
- First 125 characters shown before "more" fold — make them count
- Max caption body: 2,200 characters
- Hashtags: 20-30, placed in first comment (not caption body)
- Emojis: use tastefully, not excessively

## Format

```
[OPENING LINE — max 125 chars, hook before the fold]

[Single blank line]

[Caption body — conversational paragraphs with emojis where natural]
[Blank lines between paragraphs]

[Closing line — question or invitation to engage]

.
.
.
[COMMENT BLOCK — hashtags go in first comment]
#hashtag1 #hashtag2 ... (20-30 total)
```

## Tips
- Opening line must work without context — it's all many see
- Use emojis as visual breaks, not decoration
- The "..." section creates separation before hashtags in captions if used there
- Questions get comments; statements get likes — both have value
